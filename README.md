# 📊 Machine Learning Project - BMW Sales Prediction
Perbandingan Algoritma Linear Regression, Decision Tree, dan Random Forest dalam Prediksi Penjualan Mobil BMW

## 📌 Deskripsi
Project ini bertujuan untuk memprediksi penjualan mobil BMW menggunakan metode regresi.

## 📂 Dataset
BMW Global Sales Dataset

## 🎯 Tujuan
Memprediksi jumlah penjualan (units_sold)

## ⚙️ Algoritma
- Linear Regression
- Decision Tree
- Random Forest

## 🔧 Tahapan
- Data Understanding
- Data Preprocessing
- EDA
- Data Splitting
- Model Building
- Hyperparameter Tuning
- Evaluation

## 📈 Hasil
Model terbaik: Random Forest  
R² Score: 0.91
